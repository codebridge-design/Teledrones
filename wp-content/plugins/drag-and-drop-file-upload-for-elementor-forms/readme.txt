Allows you to add powerful Drag & Drop or choose Multiple Files Uploading area to your Elementor Forms.

== Description ==

[youtube https://www.youtube.com/watch?v=5JPfFuSDpSM]

**DEMO**: <https://demo.add-ons.org/demo-drag-and-drop-fields/>

Allows you to add powerful Drag & Drop or choose Multiple Files Uploading area to your Elementor Forms. You can provide your website visitors with a seamless and user-friendly file upload experience, allowing them to submit multiple files as part of your forms easily.


== Features ==
- **Easy to use**
- **Dropbox**: Save files to Dropbox
- **Image Preview**: Show Thumbnail for images
- **File Type Validation**
- **File Size Validation**
- **Limit number of files Upload**
- **Send files as email attachment or as a links**
- **Support multiple languages**
- **Mobile Responsive**

**Download Pro Version**: <https://add-ons.org/plugin/drag-and-drop-file-upload-for-elementor-forms-pro/>
**Document**: <https://add-ons.org/document-elementor-drag-and-drop-multiple-file-upload/>


== Installation ==
**Normal installation**

1. Download the drag-and-drop-file-upload-for-elementor-forms.zip file to your computer.
2. Unzip the file.
3. Upload the `drag-and-drop-file-upload-for-elementor-forms` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. **Document**: <https://add-ons.org/document-elementor-drag-and-drop-multiple-file-upload/>
